# AI Vault Universal v1.4

这是基于你当前满意版本继续升级的一版，主体交互不变，但新增了更像正式产品的工作台能力：

- **多模型适配**：ChatGPT / Claude / Gemini / Grok / DeepSeek / Kimi / 豆包 / Perplexity / Poe / Copilot / Mistral
- **上下文胶囊**：轻压缩 / 续聊包 / Skill 候选
- **导出中心**：Markdown / PDF / JSON
- **AI 整理**：让任意 AI 帮你输出分组与标签 JSON，再合并回本地知识库
- **发送行为增强**：AI 页面支持 Enter 换行、Ctrl(⌘)+Enter 发送；大段粘贴时提醒先做上下文胶囊

## 这版的核心升级

### 1. 全站通用层 + 站点增强层
通用层能力：
- 保存摘录
- 标签 / 文件夹 / Skill 包
- 个人信息片段库
- `/触发词` 插入
- 上下文胶囊
- 导出中心
- AI 整理

站点增强：
- 更强的“保存最后一条 AI 回复”
- AI 页面优先按问答节点导航
- 借当前 AI 总结 Skill 包 / 上下文胶囊

### 2. 上下文胶囊
可从当前选中、最后一轮、最近 3 轮、全部对话 / 页面生成：

- **轻压缩**：适合省 token
- **续聊包**：适合换模型继续聊
- **Skill 候选**：适合后续沉淀为 Skill 包

支持两种生成方式：
- **规则生成**
- **借当前 AI 总结**

### 3. 导出中心
支持导出：
- 当前选中
- 当前页面
- 当前对话 / 主内容
- 最后一条 AI 回复
- 全部知识库

导出格式：
- Markdown
- PDF（本地打开排版页后保存）
- JSON

### 4. AI 整理
插件会生成一段整理提示词，你可以发给任意 AI，让它返回：

```json
{
  "updates": [
    {
      "id": "原始id",
      "folder": "新的分组",
      "tags": ["标签1", "标签2"]
    }
  ]
}
```

再导入并“合并”到本地数据里。

### 5. 发送行为
在 AI 页面：
- Enter：换行
- Ctrl/⌘ + Enter：发送
- 粘贴超长文本：提醒先做“上下文胶囊”

## 仍然保留
- 小号竖向可拖拽悬浮入口
- 侧边栏导航 / 目录
- Skill Studio
- 长期片段 / Skill 包 / 个人信息
- JSON / Markdown 导出
- `/触发词` 插入

## 安装
1. 打开 `chrome://extensions` 或 `edge://extensions`
2. 开启开发者模式
3. 点击“加载已解压的扩展程序”
4. 选择本项目文件夹

## 说明
- 所有数据默认保存在 `chrome.storage.local`
- 不会自动上传到任何服务器
- PDF 导出走浏览器本地打印能力
- 部分站点的 DOM 结构会变化，若某一站点后续改版，可能需要微调选择器
